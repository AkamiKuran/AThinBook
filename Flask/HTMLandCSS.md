## CSS Animation

```css
<style>
@keyframes move {
    0% : {
        opacity: 1;
        height: 100%
    }
    50% : {
        opacity: 0;
        height: 100%
    }
    100% : {
        opacity: 0;
        height: 0px
    }
}
</style>

h1 {
    position: relative;
    animation-name: move;
    animation-duration: 3s;
    animation-fill-mode: forwards;
    animation-iteration-count: infinite;
}
```
